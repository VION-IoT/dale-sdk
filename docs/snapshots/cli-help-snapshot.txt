Description:
  dale — develop and publish Dale LogicBlock libraries

Usage:
  Vion.Dale.Cli [command] [options]

Options:
  -o, --output <json|table>  Output format [default: table]
  --project <project>        Path to .csproj file
  --verbose                  Show detailed output
  -?, -h, --help             Show help and usage information
  --version                  Show version information

Commands:
  new <name>  Create a new LogicBlock library project
  build       Build the project
  test        Run tests
  dev         Start the DevHost with web UI
  list        Show project info (logic blocks, contracts, properties, etc.)
  add         Add elements to a LogicBlock project
  pack        Pack project into a NuGet package
  upload      Pack and upload library to Vion Cloud
  login       Authenticate with Vion Cloud
  logout      Clear stored credentials
  whoami      Show current authenticated identity
  config      Manage CLI configuration

## dale upload
Description:
  Pack and upload library to Vion Cloud

Usage:
  Vion.Dale.Cli upload [options]

Options:
  --client-id <client-id>          Keycloak client ID (for CI/non-interactive auth)
  --client-secret <client-secret>  Keycloak client secret (for CI/non-interactive auth)
  --release-notes <release-notes>  Release notes for this version
  -e, --environment <environment>  Target environment (overrides stored config)
  --integrator-id <integrator-id>  Integrator ID (overrides stored config)
  --skip-duplicate                 Treat 409 Conflict (version already exists) as success
  -?, -h, --help                   Show help and usage information
  -o, --output <json|table>        Output format [default: table]
  --project <project>              Path to .csproj file
  --verbose                        Show detailed output
