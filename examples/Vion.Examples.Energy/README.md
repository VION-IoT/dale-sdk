# Vion.Examples.Energy

A Vion IoT Library template for creating LogicBlocks.

## Getting Started

1. **Set the startup project:**
   - In Visual Studio Solution Explorer, right-click `Vion.Examples.Energy.DevHost`
   - Select **"Set as Startup Project"**

2. **Run the DevHost:**
   - Press `F5` to start debugging
   - The browser should open automatically at `http://localhost:5000`

3. **Develop your LogicBlocks:**
   - Add your LogicBlock implementations in the `Vion.Examples.Energy` project
   - Register them in `DependencyInjection.cs`
   - Configure them in `Vion.Examples.Energy.DevHost/Program.cs`
